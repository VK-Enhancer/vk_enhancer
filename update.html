VK Enhancer обновлён до версии 3.1.4

Версия: 3.1.4 Release

18 апреля 2024
🆕 Что нового:
  • Исправлена ошибка при которой удалялись посты с любым источником
  • 💬 Теперь любое голосовое сообщение можно скачать - для этого нажмите на кнопку скачивания рядом с голосовым сообщением
  • Теперь функция изменения прозрачности блоков так же сохраняется при сохранении настроек в файл и обновляется при загрузке настроек из файла
  • Улучшена загрузка настроек из файла

Версия: 3.1.4 PreRelease

17 апреля 2024
🆕 Что нового:
  • Исправлена ошибка при которой классический интерфейс мессенджера некорректно работал с прозрачностью блоков
  • Добавлена новая функция — "Показывать результаты опросов". По умолчанию результаты опроса скрыты до выбора ответа
  • 💬 Теперь можно отправить любое аудио, как голосовое сообщение. Для этого перейдите в прикрепление вложений и нажмите на "Голосовое"
  • Удалены серверные сообщения. Вместо них теперь кнопка разворота расширения в отдельное окно
  • Добавлено сохранение настроек в файл и их восстановление из файла
  • Удалена старая проверка версий - она больше не требуется
  • 💬 Теперь если пользователь вступил/покинул/был исключен из беседы и включена функция "Скрыть имена и аватарки" — его имя будет скрыто
  • Расширение появилось в интернет-магазине Chrome. Теперь расширение устанавливать удобно, а так же, обновления будут устанавливаться автоматически(способ установки через github будет доступен)
  • В само расширение добавлены ченджлог и помощник при установке

Версия: 3.1.2 Release

16 апреля 2024
🆕 Что нового:
  • 💬 Улучшен классический интерфейс нового мессенджера
  • Добавлена ссылка на чат VK Enhancer в раздел "Информация"

Версия: 3.1.1 Release

15 апреля 2024
🆕 Что нового:
  • 💬 Не работал переход по кнопкам в бургер-меню. Все кнопки обновляли страницу. Это исправлено
  • 💬 Пара небольших исправлений касательно классического интерфейса нового мессенджера

Версия: 3.1.0 Release

15 апреля 2024
🆕 Что нового:
  • Добавлена возможность не помечать сообщения как прочитанные(нечиталка)
  • Добавлена возможность не отображать набор текста(неписалка)
  • 💬 Оптимизирован эмодзи-хотбар. Теперь с включенным хотбаром не будет лагов
  • 💬 Добавлен классический дизайн интерфейса для нового мессенджера. Кнопка переключения интерфейса находится в бургер-меню в новом мессенджере
  • 💬 Исправлен баг при котором загружалось два разных интерфейса мессенджера
  • 💬 Теперь новый мессенджер оптимизирован и практически не вызывает лагов
  • В очередной раз добавлены фотоаппараты там, где их убрали вк

Версия: 3.0.0 Release

8 апреля 2024
🆕 Что нового:
  • 💬 Активировали полный дизайн нового мессенджера ВК
  • Переписали некоторые функции для оптимизации расширения
  • Функция отключения звонков полностью переработана. Возвращен режим "Не беспокоить" с возможностью исходящих звонков
  • Пофикшена функция возвращения фотоаппаратов
  • В расширение добавлен удобный таббар, теперь функции не будут в куче
  • Теперь левое меню будет красивее при выборе кастомного фона
  • Знайте своих разработчиков! Разработчики расширения будут ждать вас во вкладке "Информация"
  • 💬 Добавлена функция изменения просмотрщика медиа
  • 💬 Оптимизирован эмодзи-хотбар
💬 - только с новым дизайном мессенджера
