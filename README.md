## Функции расширения
```
 * Возможность убрать значки
 * Возможность убрать реакции
 * Возможность добавления стикеров во вложения
 * Возможность узнать id пользователя или группы
 * Кастомизация сайта - сделайте VK таким, как считаете нужным!
```
